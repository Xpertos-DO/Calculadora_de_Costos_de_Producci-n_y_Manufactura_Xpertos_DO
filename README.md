# Calculadora_de_Costos_de_Producci-n_y_Manufactura_Xpertos_DO
Calculadora de costos de producción y manufactura para emprendedores. Estima de forma sencilla el costo de electricidad, gas y mano de obra por lote o producto, ayudando a los pequeños negocios a conocer y controlar sus gastos de producción.
